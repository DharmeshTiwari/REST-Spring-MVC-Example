Steps to test/run this project.

open the Eclipse Neon.

Go to File menu click on Import -> Existing Maven Project and give the download location of project.

It will take some time to import all dependencies.

Right click on the project and choose to run as Tomcat Server.

Once Tomcat run successfully then open the Soap UI or Post man and test with below URL

http://localhost:8080/SpringRESTFulExample/adddetails/
